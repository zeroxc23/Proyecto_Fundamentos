Sistema de Inventariado - Iventex

Planteamiento del Problema

Muchas pequeñas y medianas empresas usan métodos manuales para el inventario, generando errores, desorganización y retrasos. Las soluciones existentes suelen ser costosas o complejas. Se requiere una herramienta web sencilla y accesible.

Justificación

Inventex responde a la necesidad de una solución económica y fácil de usar. Mejora la eficiencia, reduce errores y facilita la gestión del inventario en pequeñas y medianas empresas.

Objetivo General

Desarrollar una aplicación web que permita gestionar de manera eficiente y segura el inventario de productos en pequeñas y medianas empresas.

Objetivos Específicos

- Implementar funcionalidades para la actualización y eliminación de productos.
- Diseñar una interfaz web intuitiva y accesible para los usuarios.
- Garantizar la seguridad de cada usuario implementando un sistema de autenticación y autorización diferenciando los accesos por el tipo de usuario.

Metodología

Se emplea el modelo en cascada (Waterfall), con fases secuenciales: recolección de requisitos, diseño, Implementación (desarrollo), pruebas, implementación y mantenimiento.

Cronograma de Actividades

1. Requisitos: 2 Días
2. Diseño: 5 Días
3. Desarrollo: 5 Días
4. Pruebas: 4 Días
5. Implementación y entrega: 4 Días
6. Mantenimiento : 2 Días
